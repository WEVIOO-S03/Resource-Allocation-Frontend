# Frontend Repository

This repository contains the React frontend application that connects to the Symfony backend.

## Prerequisites

- Node.js 
- npm or Yarn
- Backend API running

## Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/WEVIOO-S03/Resource-Allocation-Frontend.git
   cd Resource-Allocation-Frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   # OR
   yarn install
   ```

## Running the Application

### Development Mode
```bash
npm start
# OR
yarn start
```
The development server will start at `http://localhost:3000` (or another port if 3000 is in use).


## Accessing the Application

The application will be available at:
```
http://localhost:3000/Auth
```
Now you can create a user and login as admin to grant them access rights for the project so they can login
